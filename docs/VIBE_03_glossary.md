# 바이브 용어사전 — 코알못을 위한 기술 용어 해설 시스템

> **핵심 원칙**: 정의(Definition)가 아니라 비유(Analogy)로 설명한다.
> "API는 인터페이스입니다" → ❌
> "API는 식당 주문서예요" → ✅

- **타겟**: 기술 용어가 낯선 코알못 바이브코더
- **방향**: 한국 일상 소재 비유로 중학생도 이해 가능하게
- **연동**: 에러 응급실, 오늘의 챌린지, 바이브 레벨 시스템과 연결
- **관련 문서**: `VIBE_02_playground.md`, `VIBE_01_github_content.md`

## 용어 데이터 구조 (모든 기능의 기반)

모든 기능은 하나의 용어 데이터를 공유한다. DB가 아닌 JSON 파일로 관리.
어드민이 직접 편집 가능하고, 개발 복잡도가 낮다.

### 일정 선행조건 (중요)

`glossary.json` MVP 20개 작성은 VIBE_03 개발의 선행조건이다.
개발 일정(Week 4-5)과 분리해 어드민이 Week 1-2부터 병행 작성한다.

- 이유: 데이터가 없으면 검색/카테고리/연관용어/퀴즈/번역 연동 테스트가 막힘
- 완료 기준: Week 2 종료 전 MVP 20개 + 비기술자 1인 이해도 검수 완료

```json
// src/data/glossary.json
[
  {
    "id": "api",
    "term": "API",
    "emoji": "🍽️",
    "category": "기본개념",
    "one_liner": "두 앱이 서로 대화하는 방법",
    "analogy": "식당에서 주문서 같은 거예요. 손님(내 앱)이 원하는 걸 적어서 주방(다른 앱)에 전달하면, 주방에서 음식(데이터)을 만들어서 줘요.",
    "when_appears": "\"Claude API로 만들었어요\" → Claude라는 주방에 내 앱이 주문하는 것",
    "related": ["엔드포인트", "JSON", "요청"],
    "level": "beginner"
  }
]
```

### 카테고리 분류

| 카테고리 | 설명 | 예시 용어 |
|----------|------|---------|
| `기본개념` | 가장 자주 나오는 개념 | API, 서버, 클라이언트 |
| `터미널` | 명령어 관련 | pnpm, git, 포트, 환경변수 |
| `파일구조` | 프로젝트 파일 관련 | node_modules, .env, package.json |
| `배포` | 서비스 올리기 관련 | 배포, 도메인, HTTPS |
| `AI도구` | 바이브코딩 도구 관련 | MCP, 프롬프트, 토큰 |
| `에러` | 자주 나오는 에러 유형 | 404, 500, CORS |

```python

## Feature 4. 오늘의 용어 카드 🃏

### 컨셉

매일 용어 하나. 읽으면 XP 획득. 퀴즈 맞추면 추가 XP.
오늘의 챌린지와 같은 위치에 나란히 노출해서 시너지.

### UI 설계

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📅 오늘의 용어                🔑 D-18:22:41

  환경변수 (Environment Variable)

  [카드 보기 →]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
클릭하면 펼쳐짐:

  🔑 한 줄 요약:
  "코드 밖에 따로 보관하는 비밀 정보"

  🏠 실생활 비유:
  집 금고 비밀번호를 일기장에 쓰진 않잖아요?
  API 키, 비밀번호도 코드에 직접 쓰면 위험해요.
  그래서 .env 파일에 따로 보관하고,
  GitHub에 올릴 때 이 파일은 빼고 올려요.

  💡 오늘 챌린지에 나오는 용어예요!

  [✅ 이해했어요 +5 XP]   [🃏 퀴즈 풀기 +10 XP]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 핵심 원칙

1. **정의가 아니라 비유** — 사전적 설명은 오히려 혼란을 줌

3. **한 줄로 끝내기** — 비유는 한 줄, 부연설명은 선택
4. **언제 나오는지** — 추상적 정의보다 "이럴 때 나와요"가 더 유용
5. **데이터는 하나** — glossary.json 하나로 5개 기능 모두 구동
