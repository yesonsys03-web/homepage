# B-4 편집 UX 가드레일 명세

## 목적

편집-저장-배포 전 단계에서 품질 하한선을 강제한다.

## 문서 상태

- 상태: Draft v1
- Sprint: 2
- 선행: B-2
- 후행: C-2

## 검증 카테고리

### Blocking (저장/배포 차단)

- 필수 필드 미입력
- URL 형식 오류
- 이미지 src 없음
- 블록 order 중복

### Warning (배포 가능, 경고 표시)

- 텍스트 권장 길이 초과
- 이미지 alt 누락
- CTA 라벨 모호 표현

## 체크리스트 항목

- 콘텐츠 완전성
- 접근성(alt/링크 라벨)
- 정책/금칙어 위반
- 미리보기 검수 여부

## 출력 형식

- 필드 오류: inline 메시지
- 섹션 오류: 섹션 배지
- 글로벌 요약: 상단 요약 패널
